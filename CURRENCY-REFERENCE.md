## Special Characters for Symbols

If you need to type special currency symbols:

### Windows:
- Euro (€): Alt + 0128
- Pound (£): Alt + 0163
- Yen (¥): Alt + 0165
- Rupee (₹): Alt + 8377

### Mac:
- Euro (€): Option + Shift + 2
- Pound (£): Option + 3
- Yen (¥): Option + Y
- Rupee (₹): Option + 8377

### Copy-Paste Ready Symbols:
```
$ € £ ¥ ₹ ₽ ₩ ₪ ₱ ฿ ₫ ₡ ₵ ₦ ₨ ₴ ₸ ₺ ₼ ₾ CHF kr Kč Ft lei
```

## Tips for Currency Management

1. **Update Regularly**: Exchange rates fluctuate daily
2. **Round Numbers**: Consider rounding to .00 or .99 for cleaner pricing
3. **Monitor Major Pairs**: USD/EUR, USD/GBP, EUR/GBP
4. **Consider Margins**: Add 2-3% to exchange rate to cover fluctuations
5. **Test Thoroughly**: Always test conversions before going live

## Common Multiplier Calculations

If your default is USD and current rate is 1 USD = X:

- **1 USD = 0.85 EUR** → EUR Multiplier: `0.85`
- **1 USD = 110 JPY** → JPY Multiplier: `110`
- **1 USD = 1.35 AUD** → AUD Multiplier: `1.35`

If your default is EUR and current rate is 1 EUR = X:

- **1 EUR = 1.18 USD** → USD Multiplier: `1.18`
- **1 EUR = 130 JPY** → JPY Multiplier: `130`
- **1 EUR = 0.86 GBP** → GBP Multiplier: `0.86`

---

**Note**: Exchange rates are examples and change daily. Always verify current rates before setting multipliers.
